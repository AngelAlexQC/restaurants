## About Restaurants

It is a restaurant application developed in Laravel for educational purposes and for those who want to start with the development of their restaurant application either for portfolios or for a personal project.
The documentation is being written by the developer of the app (I am not an expert), I will try to be as explicit as possible to understand why this application is being developed

### Init with Laravel Sail (needs Docker installed).
```bash
composer install
./vendor/bin/sail up -d
```
